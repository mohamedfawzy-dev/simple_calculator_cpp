# 🧮 Simple Calculator (C++)

A simple yet functional console-based calculator built using C++.
آلة حاسبة بسيطة لكنها عملية باستخدام لغة ++C من خلال الـ Console.

---

## 💡 About | فكرة المشروع

This project demonstrates basic programming concepts such as functions, loops, and user interaction.
المشروع بيطبق أساسيات البرمجة زي الدوال (Functions)، الحلقات (Loops)، والتعامل مع المستخدم.

---

## ⚙️ Features | المميزات

* Perform basic operations (Add, Subtract, Multiply, Divide)

* Clean console interface

* Continuous usage with loop

* Handles division by zero

* تنفيذ العمليات الأساسية

* واجهة Console بسيطة ومنظمة

* يشتغل بشكل مستمر (loop)

* التعامل مع القسمة على صفر

---

## ▶️ How to Run | طريقة التشغيل

1. Compile the code using any C++ compiler

2. Run the program

3. Enter numbers and choose operation

4. اعمل Compile للكود باستخدام أي Compiler C++

5. شغّل البرنامج

6. أدخل الأرقام واختار العملية
